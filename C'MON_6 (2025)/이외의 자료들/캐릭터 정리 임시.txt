## 주인공

**이름 : 아레인 (임시)**

**캐릭터 특징**
  직업 : 농부
  기본적인 스토리 : 영주가 마을 사람들을 제물로 바치는 의식을 하면서 삶이 무너짐. 저항 끝에 영주를 쓰러뜨리고, 용을 없애기 위한 여정에 나섬.

아레인은 어릴 적 부모에게 버림받은 아이였고, 당시 고아였던 그를 지금의 아버지가 데려가 키우며 떠돌이 생활을 하게 됨. 아버지는 체술, 지식, 생존 등 다방면에서 뛰어난 인물이었으나 어떤 사건으로 위기에 처하고, 아레인에게 붉게 빛나는 목걸이를 남기고 사라짐. 이후 마을 사람들의 보살핌을 받으며 농부로 자라났으나, 영주의 횡포로 마을이 무너져가자 지켜야겠다는 신념으로 반란을 시작하게 됨.

**세부 사항**
  성별 : 남성
  나이 : 20대
  체형 : 마른 듯 하지만 단련된 체형
  머리 색 : 검정
  눈 : 푸른색
  옷 : 농부의 작업복 (해지고 낡았지만 손때 묻은 옷)
  상처 : 농사와 싸움으로 인한 상처들이 몸에 남아 있음
  아이템 : 아버지가 남겨준 붉은 빛의 목걸이 (후반부 결전에서 중요하게 사용)
  성격 : 성실하고 순박하며, 내면에는 뜨거운 열정을 품고 있음

-------------------------------

## 영주

**캐릭터 특징**
  직업 : 마을의 지배자
  기본적인 스토리 : 탐욕스러운 권력자. 자신의 권력과 부를 위해 사람들을 고대 용에게 제물로 바침

영주는 원래 평범한 귀족이었지만 용과 관련된 서적과 사제의 가르침을 접하면서 점차 타락하게 됨. 그는 권력 유지를 위해 마을 사람들을 희생시키는 것도 주저하지 않음. 사람들 앞에서는 두려움과 권위를 내세우지만 내면적으로는 용의 힘에 대한 강한 집착과 공포심을 동시에 지니고 있음.

**세부 사항**
  성별 : 남성
  나이 : 40대 후반
  체형 : 뚱뚱하고 위압적인 체형
  머리 색 : 은색 (권력자로서의 위엄을 과시하려 염색한 것)
  눈 : 검붉은 색 (탐욕의 상징)
  옷 : 비단과 금장 장식으로 치장된 사치스러운 의복
  무기 : 장검 + 갑옷 (무거운 갑옷을 입고 기사단과 함께 싸움)
  성격 : 탐욕스럽고 잔혹, 사람을 도구로 여김

-------------------------------

## 용의 기사단

**캐릭터 특징**
  직업 : 영주의 무력 집단
  기본적인 스토리 : 영주를 보좌하며 마을을 억압하는 역할. 대부분은 권력과 공포 때문에 영주를 따름.

기사단은 원래 마을을 지키던 수호 기사단이었지만 영주의 압제 아래 충성할 수밖에 없는 처지가 되었다. 일부는 진심으로 충성하지만 일부는 두려움 때문에 따르며 내적으로 갈등하고 있음.

**세부 사항**
  성별 : 남녀 혼합
  나이 : 20~40대
  체형 : 전투에 단련된 근육질 체형
  무기 : 창, 검, 방패 (개개인마다 다름)
  옷 : 무거운 갑옷, 영주의 문장이 새겨진 깃발 장식
  성격 : 겉으로는 충성스럽지만 내면은 공포와 권력욕 사이에서 갈등

-------------------------------

## 사냥꾼

* **캐릭터 특징**
  직업 : 숲 속을 떠도는 방랑자
  기본적인 스토리 : 아레인이 숲을 지날 때 만나는 동료. 활을 능숙하게 다루고, 자연과 친숙하다.

사냥꾼은 세상과 일정한 거리를 두며 자유롭게 살아가던 인물이다. 그는 아레인의 진심을 보고 협력하게 되며, 전투에서 활과 지식을 바탕으로 조언을 해준다. 또한 아레인에게 아이템(포션, 화살 등)을 무료로 주거나 싼 값에 판매하면서 든든한 동반자가 된다.

* **세부 사항**
  성별 : 남성
  나이 : 20대 중후반
  체형 : 균형 잡힌 체형, 가볍고 민첩
  머리 색 : 갈색
  눈 : 녹색 (자연과 연결된 느낌)
  옷 : 가죽 조끼, 숲에 어울리는 위장 옷
  무기 : 활과 단검
  성격 : 자유분방하고 유머러스하지만, 위기 순간에는 진지해짐

-------------------------------

## 용의 사제

* **캐릭터 특징**
  직업 : 용을 숭배하는 제단의 지배자
  기본적인 스토리 : 2막 최종 보스. 인간의 피를 제물로 바치며 용의 부활을 준비한다.

사제는 오랜 세월 동안 용을 신으로 모시며 인간을 피와 영혼의 원천으로 취급했다. 그는 강력한 마법을 다루며, 전투 중에 피를 흡수해 더욱 강해진다.

* **세부 사항**
  성별 : 남성
  나이 : 50대
  체형 : 삐쩍 마른 체형, 그러나 강한 마력으로 버팀
  머리 색 : 백발
  눈 : 붉은색 (광신을 상징)
  옷 : 붉은 제복, 의식용 로브
  무기 : 지팡이, 제단에서 부리는 마법
  성격 : 광신적, 차가움, 인간의 생명을 가볍게 여김

-------------------------------

## 용

* **캐릭터 특징**
  직업 : 고대의 재앙, 최종 보스
  기본적인 스토리 : 고대부터 이어져 내려온 불멸의 존재. 불멸의 심장을 지니고 있으며, 인간을 지배하려는 상징적인 위협.

용은 단순한 괴수가 아니라, **불멸성 그 자체를 담은 고대의 심장**을 가진 존재다. 그의 힘은 세상을 불길로 집어삼키며, 인간의 탐욕과 두려움에 의해 반복적으로 불려난다.

* **세부 사항**
  성별 : 무성 (상징적 존재)
  나이 : 수천 년 이상
  체형 : 거대한 날개와 불길에 덮인 용
  눈 : 타오르는 황금빛
  특징 : 불멸의 심장 / 하늘과 대지를 불태우는 불길
  성격 : 말이 없고, 단순한 파괴와 본능의 상징
